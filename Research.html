Research topics

  • Missing data in exploratory multivariate data analysis
    • Missing values in principal component analysis, multiple imputation with and in PCA
    • Missing values in multiple correspondence analysis, in multiple factor analysis
  • Exploratory data analysis: INDSCAL model, Multiple Factor Analysis, confidence ellipses in PCA
  • Sensometry: sensory analysis

I am working with Julie Josse on missing values in exploratory multivariate data analysis. We have proposed a regularized iterative PCA and a regularized interative MCA algorithm which allows users to perform PCA or MCA with missing values. We have also proposed a method that permits to perform multiple imputation in PCA.
