Packages

Je suis investi dans la création et le développement de plusieurs packages sous R.

  • FactoMineR : package dédié à l'analyse de données (ACP, AFC, ACM, AFM, classification, etc.).
  • missMDA : package qui permet de gérer les données manquantes en analyse des données (ACP, ACM ou AFM) et qui permet de faire de l'imputation multiple
  • Factoshiny : package fournissant une interface graphique de FactoMineR qui permet de modifier les graphiques de façon interactive.
  • FactoInvestigate : package proposant ine interprétation des résultas d'une ACP, AFC ou ACM obtenue via FactoMineR.
  • RcmdrPlugin.FactoMineR : package fournissant un menu déroulant de FactoMineR via l'interface Rcmdr.
  • SensoMineR : package dédié à l'analyse de données sensorielle. Il permet de décrire des produits d'un point de vue uni-dimensionnel ou multi-dimensionnel, d'évaluer la performance d'un panel, de faire de la cartographie des préférences, de traiter des données recueillies par napping ou par catégorisation, etc.
