Analyse des données

Plusieurs jeux de données sont issus du livre Analyse de données avec R (F. Husson, S. Lê et J. Pagès, 2009, PUR).
Le script R permet de refaire les analyses commentées dans le livre par un simple copié-collé.

  • Jeux de données pour l'Analyse en Composantes Principales (ACP)
    • Jus d'oranges (avec le script R)
    • Consommation des ménages (avec le script R)
    • Température dans les capitales européennes (avec le script R)
    • Génomique sur les poulets (avec le script R)
    • Dégustation de vins par un jury d'experts oenologues, par un jury d'étudiants et par un jury de consommateurs
    • Performance au décathlon
    • Pluviométrie dans les villes françaises
    • Température dans les villes françaises
  • Jeux de données pour l'Analyse des correspondances (AFC)
    • Enquête sur le travail des femmes (en 1970) (avec le script R)
    • Médailles au Jeux Olympiques (avec le script R)
    • Description de vins (avec le script R)
    • Causes de décès (avec le script R)
    • Effectif étudiant dans les universités
    • Résultats aux élections présidentielles de 2002 par région
  • Jeux de données pour l'Analyse des correspondances Multiples (ACM ou AFCM)
    • Enquête sur la consommation de thé (avec le script R)
    • Enquête sur les OGM (avec le script R)
    • Description de parfums par catégorisation (avec le script R)
    • Crédit bancaire
  • Jeux de données pour la Classification Ascendante Hiérarchique (CAH)
    • Température dans les capitales européennes(avec le script R)
    • Enquête sur la consommation de thé (avec le script R
    • Decoupage en classes d'une variable (avec le script R)

Jeux de données en sensométrie

  • Jeu de données parfums (dégustation 2014) : données QDA, données de catégorisation (fichiers csv)
  • Jeu de données parfums (données recueillies en 2013 par A. Emzivat, N. Le Cam, A. Gehin et E. Dufeil) : les publicités, le questionnaire, catégorisation sur les publicités, description des publicités par recueil classique QDA, catégorisation des parfums (séance 1, séance 2)
  • Jeu de données chocolat (description étudiant, et description par des experts)
  • Jeu de données jus d'orange
    • En 2005-06 : nom des jus d'oranges, description par recueil classique, description par un recueil par nappe
    • En 2006-07 : nom des jus d'oranges, description par recueil classique, description par un recueil par nappe
    • Données de préférences : nom des jus d'oranges, préférence
    • En 2009-10 : nom des jus d'oranges, description par recueil classique, description par catégorisation
    • En 2010-11 : nom des jus d'oranges, description par recueil classique, description par catégorisation
  • Jeu de données biscuit : nom des biscuits, description par recueil classique réalisée par les français et par les pakistanais, description par un recueil par nappe, description par un recueil par nappe et mots, données de préférences
  • Jeu de données vins : nom des vins, description par recueil classique réalisée par les experts et par deux jurys étudiants, description par un recueil par nappe, données de préférences
  • Jeu de données parfum : nom, description par recueil classique réalisée par des experts et par des étudiants, description par catégorisation, description par nappe d'étudiants et nappe de consommateurs, données de préférences.
  • Jeu de données cocktail : description par recueil classique réalisée par un recueil classique (QDA_cocktail), leur composition physico-chimique, description par un recueil par nappe, données de préférences (hedo_cocktail).
