Packages

Je suis investi dans la création et le développement de plusieurs packages sous R. Ces packages permettent de mettre à disposition les méthodes que nous développons au laboratoire et permettent de fournir aux étudiants et plus largement à la communauté scientifiques des outils gratuits pour l'analyse des données et la sensométrie.

FactoMineR

J'ai créé avec Sébastien Lê et je développe avec Sébastien Lê et Julie Josse le package FactoMineR, package dédié à l'analyse des données. Les méthodes les plus classiques sont implémentées (ACP, Analyse des Correspondances, Analyse des Correspondances Multiple (ACM ou AFCM) et l'Analyse Factorielle Multiple, classification ascendante hiérarchique) ainsi que des méthodes plus avancées (Analyse Factorielle Multiple Hiérarchique, Analyse de données mixtes, Analyse Factorielle Multiple Duale). De nombreuses méthodes permettent de décrire les résultats (description automatique des axes factorielles, description automatique des classes d'une partition, etc.). Une interface graphique est également disponible. Pour en savoir plus, voir le site internet de FactoMineR qui fournit toutes les aides pour installer FactoMineR, des aides pour les fonctions, des exemples d'utilisation. Le livre Analyse de données avec R fournit de nombreux exemples d'analyse de données traités avec FactoMineR.

Factoshiny

J'ai créé et je développe avec Pauline Vaissié le package Factoshiny qui permet d'utiliser le package FactoMineR à l'aide d'une interface graphique, et qui permet aussi de modifier les graphiques de façon interactive. Ce package est très utile pour optimiser ces graphiques avant de les diffuser.

missMDA

J'ai créé avec Julie Josse le package missMDA qui permet de gérer les données manquantes en analyse des données (ACP, ACM ou AFM). Ce package permet également de faire de l'imputation multiple par ACP. Ainsi, cela permet d'avoir une distribution de valeurs à la place d'une valeur manquante dans un tableau de données.

SensoMineR

SensoMineR est un package dédié à l'analyse de données sensorielle. C'est le premier logiciel de traitement de données sensorielles. Il permet de décrire des produits d'un point de vue uni-dimensionnel ou multi-dimensionnel, d'évaluer la performance d'un panel, de faire de la cartographie des préférences, de traiter des données recueillies par napping ou par catégorisation, etc. Une interface graphique est également disponible. Pour en savoir plus, voir le site internet de SensoMineR qui fournit toutes les aides pour installer FactoMineR, des aides pour les fonctions, des exemples d'utilisation.
